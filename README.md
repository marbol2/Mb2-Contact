# Mb2 Contact | Joomla Ajax Contact Module
Mb2 Contact is Joomla 3+ module to display contact form on your website.
# Key Features
<ul>
<li>Ajax form</li>
<li>Responsive layout</li>
<li>"Send me copy" option</li>
<li>Captha question</li>
<li>Multilingual</li>
<li>Lightweight</li>
<li>Very easy to Use</li>
</ul>
